# SubnetOps Diagram Cleanup Spec v360

This pass locks in the diagram workspace rules.

## Non-negotiable layout rules
- Keep the left pane as the primary diagram control area.
- Keep the top canvas toolbar row.
- Remove all duplicate control rows and narrative blocks from the live diagram area.
- The live diagram canvas must be the main focus of the page.

## Control wiring rules
- Left-pane View must switch the actual rendering mode.
- Left-pane Scope must change the actual framing/focus of the diagram.
- Left-pane Layers must visibly change what the canvas shows.
- Default state must be simple and quiet.

## Default visual rules
- Default labels: essential.
- Default link notes: minimal.
- Overlay details should appear only when the related layer is turned on.
- Validation/review narration should stay out of the live canvas path.
